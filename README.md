## Ringed Seal Detector ##

<p align="center">
  <img src="logo-seal.png" alt="Logo" width="200">
</p>

The ringed seal detector is an open-source tool for detecting ringed seal barks within passive acoustic data. It was trained using data from the Western Canadian Arctic, and focuses on the 100-1200 Hz (low frequency) range. Detection outputs are compatible with Raven Sound Analysis Software.

Installation and tutorial here: 
https://ringed-seal-detector-docs.readthedocs.io/en/latest/
